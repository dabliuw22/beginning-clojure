# beginning-clojure

Requirements:
    * JDK >= 1.8
    * Clojure
    * Leiningen
    * Docker
    * Docker Compose

1. Create project:
    ```
    lein new app beginning-clojure
    ```
    
2. Run Main:
    ```
    lein run
    ```
    
3. Run rEPL:
    ```
    lein repl
    ```
   
4. Run Test:
    ```
    lein test
    ```
